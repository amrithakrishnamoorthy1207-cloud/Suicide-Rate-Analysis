# Suicide-Rate-Analysis
This project analyzes suicide rate trends in India using data from the World Health Organization (WHO).

## Objectives
- Explore suicide trends over time
- Compare suicide rates between males and females
- Analyze suicide patterns across age groups

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset Source
World Health Organization (WHO)

https://data.who.int/indicators/i/F08B4FD/16BBF41?m49=356
